# AutoMapperAI

AutoMapperAI es un proyecto de investigación cuyo objetivo es generar automáticamente mapas (.sm) para juegos de ritmo como StepMania y BeatX utilizando Inteligencia Artificial.

## Objetivos

- Leer archivos .sm
- Analizar audio
- Extraer espectrogramas
- Entrenar modelos de IA
- Generar nuevos charts automáticamente
- Exportar nuevamente a formato .sm
