# Dataset

El dataset contiene canciones provenientes de packs públicos compatibles con StepMania.

Estructura:

raw/
sample/
processed/
features/

raw:
Canciones originales.
dataset/raw/
    2013 Anime Pack/
    Anime Pack 4/
    ...
    
processed:
Versiones procesadas.

features:
Espectrogramas, embeddings y demás características.

sample:
Canciones utilizadas para pruebas.
